# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.6.x   | :white_check_mark: |
| < 0.6   | :x:                |

## Reporting a Vulnerability

This project is a CLI music composition tool and does not handle network requests
or sensitive data. If you find a security issue, please open an issue with the
"security" label or contact the maintainer directly.

Do not open public issues for critical vulnerabilities.
